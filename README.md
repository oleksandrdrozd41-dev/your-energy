# YourEnergy

Vanilla JS web application (SPA) for browsing workout categories and exercises using public API.

## Features
- Home and Favorites pages (hash routing)
- Quote of the day (cached in localStorage by date)
- Filters: Muscles / Body parts / Equipment
- Categories → Exercises list (search + pagination)
- Exercise details modal (add/remove favorites)
- Rating modal (send rating)
- Subscribe form (email validation + API request)
- Favorites stored in localStorage
